🔬 Research Experience

Learning and Optimization for Machine Intelligence (Optima) Research Group

June 2025 – Present
Research Assistant, Department of Computer Science, City University of Hong Kong
Supervisors: Zhichao LU, Assistant Professor & Qingfu ZHANG, Chair Professor, IEEE Fellow.

Project 1: EvoEngineer — Automated Code Evolution for GPU Kernels

  • Co-authored EvoEngineer (arXiv:2510.03760), a framework that uses LLMs to automate the evolutionary optimization of CUDA kernels through iterative search in code space.
  • Contributed to building the code evolution pipeline architecture, participating in the design of the iterative LLM-based search-and-refinement workflow for progressive CUDA kernel optimization.
  • Built the end-to-end experimental infrastructure integrating KernelBench, covering automated compilation checks, correctness verification against reference PyTorch outputs, and GPU timing, enabling reproducible evaluation across a suite of real-world deep learning kernels.
  • Contributed to manuscript writing and figure preparation.

Project 2 (Ongoing): Training-Free Surrogate Model for Spiking Neural Architecture Search

  • Developed a multi-level feature extraction framework for SNN architectures, combining zero-cost proxies, code-level component statistics, structural shape-trace features, and a novel SNN-specific firing-rate metric, all extracted without any training.
  • Pioneered a fusion of 21 feature extractions with TabPFN, surpassing all traditional NAS zero-cost proxy methods and classic ML models on performance prediction for Wikitext2 and ListOps.
  • Designed an online learning loop where the surrogate continuously improves as the NAS process accumulates labeled architectures, significantly reducing the cost of architecture evaluation.
